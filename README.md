# -hospital-management-
Hospital Management System An attractive and responsive Hospital Management System (HMS) built using HTML, CSS, and JavaScript. It provides modules for patient registration, appointment booking, and billing, making it a simple yet effective demo project.
Features
Patient registration: Register new patients with details like name, age, and condition.

Appointment booking: Schedule appointments with doctors and select dates.

Billing system: Generate bills quickly with patient name and amount.

Responsive design: Works seamlessly across desktop and mobile.

Interactive alerts: Provides instant feedback for actions.
⚙️ Setup Instructions
Clone or download this repository.

Open index.html in your browser.

Use the forms to register patients, book appointments, and generate bills.
📌 Technologies Used
HTML – Structure of the system

CSS – Styling and layout

JavaScript – Interactivity and form handling
